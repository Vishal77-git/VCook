# VCook - Sustainable Cooking App

This is a standalone, zero-dependency version of the VCook application.

## How to Run Locally

1.  **Directly**: Simply double-click the `index.html` file in your browser.
    *   *Note*: This version uses a single-file structure and CDN-based Tailwind CSS to ensure it works instantly without any setup.

2.  **With a Local Server (Recommended for Developers)**:
    *   If you have Node.js installed, you can run:
        ```bash
        npx serve .
        ```
    *   Or use the "Live Server" extension in VS Code.

## Features
- **Browse Recipes**: Discover sustainable, zero-waste recipes.
- **Search**: Filter by ingredients or keywords.
- **Submit Recipe**: Share your own sustainable culinary creations.
- **Responsive Design**: Works on mobile and desktop.
- **No Dependencies**: All logic and styles are contained within `index.html`.
